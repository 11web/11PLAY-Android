# 11web Android APK

App name: 11web  
Package: com.oneoneweb.app  
Website: https://oneoneweb.github.io/11Web/

## GitHub Actions Build

1. Upload all files to a new GitHub repository.
2. Open the repository on GitHub.
3. Go to Actions.
4. Run "Build 11web APK".
5. Download artifact: 11web-debug-apk.
6. Extract the artifact ZIP and install app-debug.apk.
