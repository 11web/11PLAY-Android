# 11PLAY Android APK

App name: 11PLAY
Package: com.elevenplay.app
Website: https://11web.github.io/11play/

## GitHub Actions Build

1. Upload all files to a new GitHub repository.
2. Open the repository on GitHub.
3. Go to Actions.
4. Run "Build 11PLAY APK".
5. Download artifact: 11PLAY-debug-apk.
6. Extract the artifact ZIP and install app-debug.apk.
